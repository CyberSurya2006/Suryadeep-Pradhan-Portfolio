**Suryadeep Pradhan — Developer Portfolio**

An interactive developer portfolio showcasing my projects, skills, and development journey through immersive UI, 3D visuals, and smooth animations.
Built with modern web technologies to create a visually engaging and highly interactive user experience.

*🚀 Tech Stack*


**Framework & Core**
  - Next.js (App Router)

  - React

  - TypeScript


**3D Graphics**
  - Three.js

  - React Three Fiber

  - React Three Drei


**Animations**
  - GSAP

  - GSAP ScrollTrigger

  - Styling

  - Tailwind CSS

  - CSS Variables


**UI**
  - shadcn/ui

  - Lucide


**Other**
- Canvas API (2D particle effects)

- Custom cursor

- Native HTML5 Audio API

- Responsive mobile-first design


*✨ Features*

  - Immersive 3D visual elements

  - Scroll-driven animations

  - Interactive text hover effects

  - Unique developer journey section

  - Custom cursor and micro-interactions

  - Smooth page transitions

  - Fully responsive design


*🌐 Deployment*

This portfolio is deployed using Vercel for fast global delivery and seamless integration with Next.js applications.
